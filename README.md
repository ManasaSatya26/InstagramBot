# Instagram DM Bot

A Node.js application that listens for Instagram comments using Meta Webhooks and triggers automated responses based on keywords.

## Features

- Real-time comment detection
- Keyword-based automation
- Meta Graph API integration
- Webhook event handling

## Tech Stack

- Node.js
- Express.js
- Meta Graph API
- Webhooks
- Ngrok

## Note

This project is currently configured for development/testing purposes and requires Meta App Review for full production functionality.
